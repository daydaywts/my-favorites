# 图标替换说明

网站图标全部放在这个文件夹，可以直接用自己的 SVG 文件覆盖同名文件，不需要修改页面代码。

| 文件 | 使用位置 |
| --- | --- |
| `site.svg` | 浏览器标签页图标、网站主图标 |
| `menu.svg` | 手机端菜单按钮 |
| `search.svg` | 顶部与搜索框图标 |
| `collection.svg` | 首页 |
| `movie.svg` | 影视 |
| `manga.svg` | 漫画 |
| `game.svg` | 游戏 |
| `tag.svg` | 标签分类 |
| `theme.svg` | 日间模式的太阳图标 |
| `moon.svg` | 夜间模式的月亮图标 |
| `atmosphere.svg` | 关于页的氛围偏好 |
| `close.svg` | 关闭按钮 |
| `quote.svg` | 引语图标 |

建议 SVG 使用 `viewBox="0 0 24 24"`，图形颜色使用黑色。深色模式会自动反转成白色。
